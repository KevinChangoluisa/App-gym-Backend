# App-gym-Backend
 Servidor para la aplicación para dietas y rutinas de ejercicio

## Installation

Requires [Node.js](https://nodejs.org/).

Install the dependencies and devDependencies and start the server.

```sh
npm install
```

## Start
```sh
npm run dev
```
## Requirements
